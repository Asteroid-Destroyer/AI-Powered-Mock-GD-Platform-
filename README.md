# AI-Based Group Discussion Mock Platform

## 🚀 Overview
An AI-powered platform that simulates group discussions using virtual participants and provides real-time feedback.

## 🧠 Features
- AI-generated GD simulation
- Speech & communication analysis
- Confidence and posture tracking
- Instant feedback system

## 🛠️ Tech Stack
- Frontend: React
- Backend: Node.js
- AI: Python (NLP)
- APIs: OpenAI, Speech Recognition

## 📂 Project Structure
frontend/ - UI  
backend/ - API server  
ai-engine/ - ML models  

## ▶️ How to Run

### Frontend
cd frontend  
npm start  

### Backend
cd backend  
node server.js  

### AI Engine
python analysis.py
